# Java 17 + PostgreSQL 18 DevContainer

## :computer: Tecnologias

- Java 17
- PostgreSQL 18
- Docker / Docker Compose

## :rocket: Como rodar

1. Abra no VS Code ou IntelliJ
2. Reopen in Dev Container

## :floppy_disk: Banco de dados

- Host: db
- Porta: 5432
- Usuário: postgres
- Senha: postgres

## :white_check_mark: Teste rápido

```bash
java -version
```

```bash
./mvnw -v
```

```bash
psql -h db -U postgres
```
